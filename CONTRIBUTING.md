# 🤝 Contributing to SamarthyaBot

Thank you for your interest in contributing to **SamarthyaBot** — India's privacy-first local AI operating system! 🇮🇳

We welcome contributions of all kinds: bug reports, feature requests, documentation, code, and translations.

---

## 📋 How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/mebishnusahu0595/SamarthyaBot.git
cd SamarthyaBot
npm install
```

### 2. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes

- Follow existing code style and conventions
- Add comments for complex logic
- Test your changes locally with `npm start`

### 4. Commit with a Descriptive Message

```bash
git commit -m "feat: add weather plugin for autonomous weather checks"
```

We follow [Conventional Commits](https://www.conventionalcommits.org/):
- `feat:` — New feature
- `fix:` — Bug fix
- `docs:` — Documentation
- `refactor:` — Code refactoring
- `chore:` — Maintenance

### 5. Push & Create a Pull Request

```bash
git push origin feature/your-feature-name
```

Open a PR on GitHub. Describe what you changed and why.
